# Quotes package
This folder contains JSON files with curated quotes in English, Hindi and Gujarati.

Structure:
- raw/
  - quotes.json           # all mixed quotes
  - quote_of_day.json
  - random_quote.json
  - english.json
  - hindi.json
  - gujarati.json
  - categories/
    - love.json
    - sad.json
    - friendship.json
    - attitude.json
    - life.json
    - motivation.json
    - breakup.json
    - alone.json
    - romantic.json
    - festival.json
    - daily.json

You can upload the `raw/` folder to your repository as-is.
